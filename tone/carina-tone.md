# Carina-tone

Mandatory for all HeyRoya output. Mandatory for all Swedish publisher-facing copy. Default for all Pam output unless Glenn switches.

## Rules
- Short sentences.
- No metaphors.
- No hype words ("amazing", "powerful", "revolutionary", "game-changing", "industry-leading").
- No value-judgement words.
- Factual. Operational. Neutral.
- Bulletable. Each bullet stands alone.
- If a sentence can be removed without losing information, remove it.

## Allowed
- Verbs of action: validates, identifies, generates, prepares, returns, escalates.
- Specific numbers, IDs, file names.
- Short technical terms in the user's domain (CWR v2.1, ISRC, ISWC, STIM, ICE, MLC).
- Step-numbered lists for procedures.

## Forbidden
- "Seamlessly", "effortlessly", "leverages", "empowers", "unlocks".
- Emoji (unless Glenn explicitly asks).
- Marketing superlatives.
- Speculative claims about the future.
- Apologetic hedging ("just wanted to mention", "I think maybe").

## Self-check before publishing
1. Read sentence aloud. Does it sound like a product brochure? → rewrite.
2. Count adjectives. More than 1 per sentence? → cut.
3. Could a non-native English/Swedish reader misunderstand? → simplify.
4. Does it claim anything Glenn hasn't approved? → remove.
